JQuery (khái niệm)

    Jquery là thư viện được xây dựng từ javascript nhằm giúp coder xây dựng ứng dụng javascript dễ dàng hơn.

    Các module chính mà của Jquery sử bao gồm.
    Ajax - xử lý Ajax
    Atributes - xử lý thuộc tính của dối tượng HTML
    Effect - xử lý hiệu ứng
    Event - xử lý sự kiện
    Form - xử lý form
    DOM - xử lý Data Object Model
    Selector - xử lý luồng lách các đối tượng HTML

Phương thức

.hide() ẩn
.toggle() ẩn và hiện
.fadeIn() hiện mờ dần
.fadeToggle() ẩn hiện mờ dần
.fadeTo() ẩn dừng lại
.stop() dừng đoạn lệnh
.val() trả về giá trị value
.attr() lấy giá trị thuộc tính
.append() chèn thêm nội dung sau tương tự .before()
.prepend() chèn thêm nội dung trước tương tự .after()
.show() hiện
.dblclick() đúp chuột
.mouseenter() hover chuột
.fadeOut() ẩn mờ dần
.sliderDow() cuộn xuống
.slideUp() cuộn lên
.slideToggle() cuộn lên xuống
.remote() xóa tất cả
.empty() xóa phần tử con
