CSS (khái niệm)

                    CSS viết tắt Cascading Style Sheets
                    CSS là ngôn ngữ mô tả phong cách tài liệu, cách hiển thị html, bố cục của nhiều trang web cùng lúc
                    ba cách để chèn css kiểu ngoài, kiểu nội bộ và kiểu nội tuyến (inline, external, internal)
                    sử dụng css để xác định kiểu cho trang web
                    bộ quy tắc css bao gồm bộ chọn(thẻ html, id, class) và khối khai báo(tên thuộc tính và giá trị)
                    màu được dùng bằng các giá trị rgb, hex, hsl, rgba, hsla
                    display: block hiển thị trên 2 hàng và không có khoảng cách, display: inline-block thì ngược lại
                    pseudo_classes (lớp giả) để định nghĩa trạng thái đặc biệt
                    pseudo_elements (phần tử giả) tạo kiểu cho phần tử chỉ định
                    gradient (phối màu) hiển thị trơn tru giữa 2 hay nhiều màu
                    Responsive làm trang web trông đẹp hơn trên tất cả thiết bị
                    SCSS là 1 processor hỗ trợ quản lý code 
                    kế thừa trong scss @extend
                    mô hình hộp: 
                    layout:
